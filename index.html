LODB.io

Experimental data publishing platform,
automatically building data driven services
and websites from Linked Open Data
schemas & vocabularies.

Simplifying data publishing, with every application contributing to a cross-domain graph of knowledge.

1. Select schemas

Select the schemas and vocabularies from the many published on the internet spanning all domains and subjects.

2. Fields and relationships

Select which fields to include from each schema, and how multiple schemas join with each other.

3. Build application

Run the build script which builds the application with a relational database, models, data validation and input forms, all auto-magically generated from the schemas themselves.

Linked By Default

All fields are part of existing, public schemas. When it comes to publishing your data there's no longer any need to retrofit internal data models to external linked open data schemas - all your data is linked by default from the outset.
